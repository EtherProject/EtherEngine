# CheckPointInCircle

> 判断点是否位于指定圆形区域内(包含圆形边界）

```lua

flag = CheckPointInCircle(point, center, radius [, border])

```

## 参数：

+ point [table]：点坐标
+ center [table]：圆心坐标
+ radius [number]：圆半径
+ border [boolean]：是否包含边界，可选，默认为 true

## 返回值：

+ flag [boolean]：是否位于区域内
