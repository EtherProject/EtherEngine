# DrawFillPie

> 在指定位置绘制无填充扇形

```lua

DrawFillPie(center, radius, startAngle, endAngle)

```

## 参数：

+ center [table]：扇形所在圆的圆心坐标
+ radius [number]：扇形所在圆的半径
+ startAngle [number]：扇形开始角度
+ endAngle [number]：扇形结束角度

## 返回值：

无

## 备注：

+ 扇形从以其所在圆心为原点的二维笛卡尔坐标系的x轴正方向开始绘制，角度的正方向为顺时针方向
