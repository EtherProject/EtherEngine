# Font:SetStyle

> 设置已加载字体的样式

```lua

font:SetStyle(style)

```

## 参数：

+ font [userdata-Font]：字体数据
+ style [table]：用以描述字体样式的表，成员可以为以下样式中的一种或多种：

    + FONT_STYLE_BOLD：粗体
    + FONT_STYLE_ITALIC：斜体
    + FONT_STYLE_UNDERLINE：下划线
    + FONT_STYLE_STRIKETHROUGH：删除线
    + FONT_STYLE_NORMAL：默认

## 返回值：

无
