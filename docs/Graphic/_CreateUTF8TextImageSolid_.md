# CreateUTF8TextImageSolid

> 使用Solid模式创建文本图像

```lua

image = CreateUTF8TextImageSolid(font, text, color)

```

## 参数：

+ font [userdata-Font]：字体数据
+ text [string]：UTF-8编码格式的文本内容
+ color [table]：用以描述文本颜色的RGBA表，各分量取值范围均为0-255

## 返回值：

+ image [userdata-Image]：成功则返回图像数据，失败则返回nil
